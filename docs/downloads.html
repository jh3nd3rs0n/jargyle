Overview

This document discusses the types of downloads for Jargyle.

Prerequisites

Jargyle requires JDK 9 or higher. It is recommended to use JDK 21 or higher to take advantage of using virtual threads instead of platform threads.

Please note that while it is encouraged to try Jargyle and report any issues, Jargyle is still a work in progress and is not yet production-ready.

Distributions

Jargyle is distributed in two distribution archive types: binary and source. Each type of distribution archive comes in multiple archive formats for you to choose.

Binary Distribution

Select a binary distribution archive and follow the installation instructions.

Link Checksum
Binary tar.bz2 archive jargyle-5.0.0-SNAPSHOT-bin.tar.bz2 jargyle-5.0.0-SNAPSHOT-bin.tar.bz2.sha512
Binary tar.gz archive jargyle-5.0.0-SNAPSHOT-bin.tar.gz jargyle-5.0.0-SNAPSHOT-bin.tar.gz.sha512
Binary zip archive jargyle-5.0.0-SNAPSHOT-bin.zip jargyle-5.0.0-SNAPSHOT-bin.zip.sha512

Source Distribution

Select a source distribution archive if you intend to build Jargyle yourself.
